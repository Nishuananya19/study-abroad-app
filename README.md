# Study Abroad App 🌍

## 📱 Overview

This is a React Native mobile application that displays a list of study abroad programs. Users can browse universities and view detailed information about each program.

## 🚀 Features

* List of universities/programs
* View details of each program
* Clean and reusable components
* Responsive UI

## 🛠️ Tech Stack

* React Native CLI
* TypeScript
* React Navigation

## 📂 Project Structure

```
src/
 ├── components/
 ├── screens/
 ├── data/
 └── navigation/
```

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/Nishuananya19/study-abroad-app.git
```

### 2. Install dependencies

```
npm install
```

### 3. Run the app

```
npx react-native run-android
```

## 📸 Screens

* Home Screen: Displays list of programs
* Details Screen: Shows full program details
